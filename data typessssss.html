JavaScript Data Types

Data Type Description Example
Number Represents integers or floating-point numbers 42, 3.14, -10
String Represents sequences of characters 'Hello', "World"
Boolean Represents true or false true, false
Undefined Variable is declared but not initialized let x;
Null Represents the absence of any value let y = null;
Symbol Represents a unique identifier Symbol('id')
BigInt Represents large integers 1234567890123456789012345678901234567890n
Object A collection of properties and values { name: 'Alice', age: 25 }
Array An ordered list of elements [1, 2, 3, 4]
Function A block of reusable code function add(a, b) { return a + b; }
